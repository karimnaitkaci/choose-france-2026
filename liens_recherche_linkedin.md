## 📌 COMMENT UTILISER CES LIENS

1. **Cliquez sur le lien** → S'ouvre directement sur LinkedIn avec les résultats de recherche
2. **Filterz les résultats:**
   - Cliquez "Personnes" (en haut)
   - Filtre par "Localisation" (si besoin: Asie, Moyen-Orient, Europe)
   - Triez par "Pertinence" (pour meilleurs résultats)
3. **Consultez les profils** → Regardez leur expérience, connections
4. **Envoyez une connexion** → Ajoutez un message personnalisé
5. **Follow-up** → Attendez 2-3 jours, puis message

## 💡 TIPS

- **LinkedIn Sales Navigator** (version payante) = meilleurs filtres & recherches avancées
- Ces recherches donnent les **profils actifs actuellement**
- Certains profils peuvent être **privés** = pas visible sans connexion
- **Contactez HR/Talent d'abord** (taux de réponse plus élevé)
- Si pas de réponse HR → Contactez **Store Managers/Directors** (volumes importants)
